# 🌐 Portfólio Profissional - Débora Cecília

Este projeto é um site de portfólio profissional desenvolvido em HTML com foco em apresentar minha trajetória acadêmica, experiências profissionais, habilidades técnicas e projetos.

O objetivo do site é reunir minhas principais experiências nas áreas de:

- Qualidade
- Processos
- Engenharia de Alimentos
- Power BI
- Python
- Análise de Dados

---

## 🚀 Tecnologias Utilizadas

- HTML
- Google Fonts
- Design Responsivo

---

## 📋 Funcionalidades

✔️ Página inicial moderna  
✔️ Seção “Quem sou eu”  
✔️ Experiência profissional  
✔️ Habilidades técnicas  
✔️ Projetos  
✔️ Layout responsivo  
✔️ Cards interativos  
✔️ Tags de tecnologias e competências  

----

## 📱 Responsividade

O site foi desenvolvido para funcionar em:
- Computadores
